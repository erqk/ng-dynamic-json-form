# 樣式

## 全域變數

以下是每一個生成的表單的全域 CSS 變數。影響範圍只限 `.ng-dynamic-json-form` class 之內。

- `--color-primary`
- `--color-primary-lighter`
- `--color-error`
- `--color-border`
- `--font-family`
- `--font-size-title`
- `--font-size-label`
- `--font-size-description`
- `--font-size-error`
- `--font-weight-title`
- `--font-weight-label`
- `--input-border-width`
- `--column-gap`
- `--row-gap`
- `--options-column-gap`
- `--options-row-gap`

我們可以將變數覆寫為新的值：

```css
.ng-dynamic-json-form {
  --color-primary: teal;
  ...;
}
```
