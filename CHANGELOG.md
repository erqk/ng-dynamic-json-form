### NgDynamicJsonFormComponent

- Remove `uiComponents` and `customValidators` input, as this makes no sense to overwrite locally.
- Rename `globalLayout...` to `layout...`.

### ErrorMessageComponent

- Use abstract class for custom component to isolate methods and variables.

### FormLabelComponent

- Use abstract class `CustomFormLabel` to implement custom component, instead of extending `FormTitleComponent`.

### FormControlConfig

- Remove `formArray`, this should be done by using custom component because there are so many possibilities that are impossible to cover.
- Remove the exposure of JSON schema.
- Unified the `config` data structure, no more “configs” key when pass using JSON.

### FormControlOptions

- No more `sourceList`, use `src`
- Rename `sourceAppendPosition` to `srcAppendPosition`
- `trigger` is rewrite and move into `src.trigger`

### Conditions

- Add prefix to the name of condition to indicate action type. (e.g. `validators.xxx`, `control.xxx`)

### Input mask

- Input mask now use imaskjs.
